## 目录说明

### frontend/

React 前端应用，负责用户界面和地图渲染。

- **components/**: UI 组件，按功能模块分类
- **hooks/**: 自定义 React Hooks，封装复用逻辑
- **stores/**: Zustand 状态管理，轻量级全局状态
- **types/**: TypeScript 类型定义，确保类型安全
- **utils/**: 工具函数，纯函数逻辑

### backend/

Python 后端应用，负责游戏逻辑和 API。

- **api/**: FastAPI 路由，RESTful API 端点
- **core/**: 游戏核心逻辑，独立于框架
- **models/**: Pydantic 数据模型，数据验证
- **services/**: 服务层，业务逻辑封装
- **ai/**: AI 决策模块，势力行为控制

### docs/

项目文档，包括策划案、技术文档、开发日志。

### scripts/

开发脚本，自动化常见任务。

### docker/

Docker 配置，容器化部署支持。
