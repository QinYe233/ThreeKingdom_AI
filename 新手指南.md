## 1. 环境准备

### 1.1 必需软件

在运行游戏之前，请确保你的电脑已安装以下软件：

| 软件 | 版本要求 | 下载地址 | 说明 |
|------|---------|---------|------|
| **Python** | 3.11 或更高版本 | [python.org](https://www.python.org/downloads/) | 后端运行环境 |
| **Node.js** | 18.0 或更高版本 | [nodejs.org](https://nodejs.org/) | 前端运行环境 |
| **Git** | 任意版本 | [git-scm.com](https://git-scm.com/) | 代码管理（可选） |

### 1.2 检查是否已安装

打开命令行（Windows 按 `Win + R`，输入 `cmd`），依次输入以下命令：

```bash
python --version
```

```bash
node --version
```

如果显示版本号（如 `Python 3.11.0` 或 `v18.0.0`），说明已安装成功。

> ⚠️ **注意**：如果提示"不是内部或外部命令"，说明未安装或未添加到环境变量，请先安装对应软件。

### 1.3 推荐配置

- **操作系统**：Windows 10/11、macOS 10.15+、Ubuntu 20.04+
- **内存**：4GB 以上
- **网络**：需要访问 AI API（如 OpenAI、DeepSeek 等）

## 2. 安装步骤

### 2.1 下载项目

**方式一：克隆仓库（需要 Git）**

```bash
git clone https://github.com/QinYe233/ThreeKingdom_AI.git
cd ThreeKingdom_AI
```

**方式二：下载 ZIP 压缩包**

1. 访问 [GitHub 仓库](https://github.com/QinYe233/ThreeKingdom_AI)
2. 点击绿色的 `Code` 按钮
3. 选择 `Download ZIP`
4. 解压到任意目录

### 2.2 安装依赖

**Windows 用户**：双击运行 `安装依赖.bat`

**Mac/Linux 用户**：

```bash
# 进入后端目录，安装 Python 依赖
cd backend
pip install -r requirements.txt

# 进入前端目录，安装 Node.js 依赖
cd ../frontend
npm install
```

> 💡 **提示**：如果下载速度较慢，可以使用国内镜像：
> ```bash
> # Python 镜像
> pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
>
> # Node.js 镜像
> npm install --registry=https://registry.npmmirror.com
> ```

## 3. 启动游戏

### 3.1 一键启动

**Windows 用户**：双击运行 `启动游戏.bat`

**Mac/Linux 用户**：

```bash
# 启动后端
cd backend
python -m uvicorn app.main:app --reload --host 0.0.0.0 --port 8000

# 新开一个终端，启动前端
cd frontend
npm run dev
```

### 3.2 访问游戏

启动成功后，打开浏览器访问：

- **游戏界面**：http://localhost:5173
- **后端 API**：http://localhost:8000
- **API 文档**：http://localhost:8000/docs

### 3.3 停止游戏

**Windows 用户**：双击运行 `停止服务.bat`

**Mac/Linux 用户**：在终端按 `Ctrl + C` 停止服务

### 4.2 功能按钮说明

| 按钮 | 功能 |
|------|------|
| **开始游戏** | 启动游戏，AI 开始自动决策 |
| **暂停** | 暂停游戏，停止 AI 决策 |
| **速度** | 调整游戏速度（慢速/正常/快速/极速） |
| **设置** | 配置 AI 模型（API Key、模型选择等） |
| **史官** | 查看历史记录和叙事日志 |
| **外交** | 查看各国之间的外交关系 |

### 4.3 地图操作

| 操作 | 说明 |
|------|------|
| **左键点击区块** | 选中区块，查看详细信息 |
| **中键拖拽** | 平移地图 |
| **滚轮** | 缩放地图 |
| **⭐ 星形标记** | 表示该区块是国家首都 |

### 4.4 区块信息

选中区块后，右侧面板会显示：

- **区块名称**：如"许昌"、"成都"等
- **归属势力**：魏/蜀/吴/中立
- **兵力**：当前驻军数量
- **人力**：可征兵人口
- **发展度**：经济水平

## 5. AI 配置

### 5.1 支持的 AI 模型

游戏支持多种 AI 模型：

| 模型 | 说明 | 推荐指数 |
|------|------|---------|
| **GPT-4** | OpenAI 最强模型 | ⭐⭐⭐⭐⭐ |
| **GPT-3.5-turbo** | 性价比高 | ⭐⭐⭐⭐ |
| **DeepSeek** | 国产大模型，性价比高 | ⭐⭐⭐⭐⭐ |
| **自定义 API** | 支持兼容 OpenAI 格式的 API | ⭐⭐⭐ |

### 5.2 配置步骤

1. 点击界面右上角的 **「设置」** 按钮
2. 选择 AI 模型类型
3. 输入 API Key（从对应平台获取）
4. 如使用自定义 API，填写 Base URL
5. 点击 **「保存」**

### 5.3 获取 API Key

**OpenAI (GPT-4)**：
1. 访问 [OpenAI Platform](https://platform.openai.com/)
2. 注册/登录账号
3. 进入 API Keys 页面
4. 点击 "Create new secret key"

**DeepSeek**：
1. 访问 [DeepSeek 官网](https://www.deepseek.com/)
2. 注册/登录账号
3. 进入 API Keys 页面
4. 创建新的 API Key

> ⚠️ **注意**：API Key 是敏感信息，请勿泄露给他人！

### 5.4 为每个势力配置不同 AI

游戏支持为魏、蜀、吴三个势力分别配置不同的 AI 模型，让它们展现不同的战略风格：

1. 在设置面板中，选择对应的势力标签
2. 分别配置各自的 AI 模型和 API Key
3. 保存后即可开始游戏

## 6. 常见问题

### Q1: 启动时提示"Python 不是内部或外部命令"

**原因**：Python 未安装或未添加到环境变量

**解决方法**：
1. 重新安装 Python，勾选 "Add Python to PATH" 选项
2. 或手动将 Python 安装目录添加到系统环境变量

### Q2: 启动时提示"npm 不是内部或外部命令"

**原因**：Node.js 未安装或未添加到环境变量

**解决方法**：
1. 重新安装 Node.js
2. 或手动将 Node.js 安装目录添加到系统环境变量

### Q3: 前端启动失败，提示端口被占用

**原因**：5173 端口被其他程序占用

**解决方法**：
1. 关闭占用端口的程序
2. 或修改 `frontend/vite.config.ts` 中的端口配置

### Q4: AI 决策时报错 "API Key invalid"

**原因**：API Key 无效或过期

**解决方法**：
1. 检查 API Key 是否正确
2. 确认 API Key 是否有余额
3. 重新生成 API Key

### Q5: 地图显示空白或区块无法点击

**原因**：地图数据加载失败

**解决方法**：
1. 检查 `docs/three_kingdoms.geojson` 文件是否存在
2. 检查后端是否正常启动
3. 刷新浏览器页面

### Q6: 游戏运行缓慢

**原因**：AI 响应速度慢或电脑配置低

**解决方法**：
1. 选择响应更快的 AI 模型
2. 调低游戏速度
3. 关闭其他占用资源的程序

### Q7: 如何重置游戏？

**解决方法**：
1. 点击「停止服务」停止游戏
2. 删除 `backend/data/game_state.json` 文件（如存在）
3. 重新启动游戏
